# AUTONIX Frontend

Frontend website untuk AUTONIX - platform trading bot otomatis.

## Teknologi yang Digunakan

- **Astro** - Framework web modern untuk membangun website yang cepat
- **TypeScript** - JavaScript dengan type safety
- **CSS** - Styling

## Struktur Proyek

```
src/
├── components/     # Komponen reusable
├── layouts/        # Layout templates
├── pages/          # Halaman website
└── ...

public/             # Static assets
```

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Build untuk production:
```bash
npm run build
```

## Fitur

- Landing page AUTONIX
- Halaman produk (EA Bot)
- Design responsive
- Performance optimized dengan Astro

Proyek ini dikembangkan untuk AUTONIX platform.
